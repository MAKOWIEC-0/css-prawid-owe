Identyfikatory — ID

Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID to skrót od identifier – oznacza unikalny identyfikator przypisany do jednego, konkretnego elementu na stronie HTML.
Jaki jest cel stosowania ID? ID pozwala na indywidualne stylizowanie i precyzyjne odwoływanie się do konkretnego elementu na stronie – zarówno w CSS, jak i w JavaScript.
Jakie jest znaczenie znaku # Znak # w CSS służy do odwoływania się do elementu na podstawie jego identyfikatora (ID). Na przykład: #nazwa_id oznacza stylowanie elementu z ID o nazwie „nazwa_id”.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? W HTML nadajemy elementowi identyfikator za pomocą atrybutu id, np.
.W CSS odwołujemy się do tego elementu przy użyciu #nazwa
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. Elementy z ID muszą być unikalne – na stronie tylko jeden element może mieć dane ID.
  2. Jeśli dany element może występować więcej niż raz, zamiast ID lepiej użyć klasy (class)
  3. Tego samego ID nie powinno się używać wielokrotnie w jednym dokumencie HTML.
